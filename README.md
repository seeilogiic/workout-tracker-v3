# Workout App (React + Vite + TS + Supabase)

White-first UI with dark accents. Includes username/password auth, bottom navigation (hidden on the auth screen), and Supabase schema for user profiles, workouts, exercises, and plans.

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`

## Supabase setup
1) Enable Email/Password sign-in and disable OAuth providers.
2) Run `supabase/schema.sql` in the SQL editor.
3) Add your keys to `.env` (see `.env.example`).
